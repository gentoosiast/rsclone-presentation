tech stack

Nest.js

  • good modular architecture out of the box
  • dependency injection pattern (inspired by Angular)
  • first-class TypeScript support
  • dozens of built-in modules that help you easily integrate with common technologies and concepts
  • higher learning curve in comparison to Express

Socket.IO

  • disconnect detection and automatic reconnect
  • rooms support out of the box
  • first class support of TypeScript
  • very good official documentation
  • none (for our use case)

the game

Если защищающийся берет карты, то напротив его иконки повяляется всплывающее сообщение, а так же подсказка справа над столом текущего игрока

the game

У активного игрока подсвечиваются карты, которыми можно ходить - больше по рангу той же масти и козырные, если он отбивается. Также он может взять карты.

the game

у активного игрока подсвечиваются карты того же ранга, что и карты на столе, если он нападающий. Или он может нажать пасс.

the game

В каждой комнате есть чат. Как видно на слайде, после раздачи карты игроков сортируюся по мастям, козырные карты в конце

the game

Если чат свернут, то приходят уведомления о сообщениях.

the game

Если на столе больше 3х пар карт, то происходит анимация расположения на столе 4-6 пар.

the game

Когда 1 из игроков скидывает карты, то он становится победителем. Если всего в игре 2 игрока, то игра заканчивается, если больше, то продолжается между игроками с картами.

the game

Победивший игрок может выйти из комнаты, игра на этом не закончится.

the game

the game
